Welcome to the Gamba API documentation. Here you will find all the information you need to interact with our API.

Explore the different endpoints below for detailed information on how to use each one.

## üéØ Endpoints

### GET /events/poolChanges

Returns transaction signatures of recent pool changes.

**Query Parameters**:

- `pool` (optional): Filter by pool ID.

**Response**:

```json
{
  "results": [
    {
      "signature": "string",
      "action": "string",
      "amount": "number",
      "user": "string",
      "token": "string",
      "pool": "string",
      "lp_supply": "number",
      "post_liquidity": "number",
      "time": "timestamp"
    }
  ]
}
```

**Example Request**:

```
GET https://api.gamba.so/events/poolChanges?pool=examplePoolId
```

---

### GET /events/settledGames

Returns settled games with various filtering and pagination options.

**Query Parameters**:

- `page` (optional): Page number for pagination.
- `onlyJackpots` (optional): Filter by jackpots.
- `creator` (optional): Filter by creator.
- `pool` (optional): Filter by pool.
- `token` (optional): Filter by token.
- `user` (optional): Filter by user.
- `orderBy` (optional): Sort results by `multiplier`, `usd_profit`, or `time`.
- `sorting` (optional): Sorting order `ASC` or `DESC`.
- `itemsPerPage` (optional): Number of items per page (1-200).

**Response**:

```json
{
  "results": [
    {
      "signature": "string",
      "wager": "number",
      "payout": "number",
      "usd_wager": "number",
      "usd_profit": "number",
      "profit": "number",
      "user": "string",
      "creator": "string",
      "token": "string",
      "jackpot": "number",
      "multiplier": "number",
      "time": "timestamp"
    }
  ],
  "total": "number"
}
```

**Example Request**:

```
GET https://api.gamba.so/events/settledGames?page=1&itemsPerPage=10&orderBy=time&sorting=DESC
```

### GET /players

Returns a list of top-performing players.

**Query Parameters**:

- `creator` (optional): Filter by creator.
- `token` (optional): Filter by token.
- `pool` (optional): Filter by pool.
- `limit` (optional): Limit the number of results.
- `offset` (optional): Offset for pagination.
- `sortBy` (optional): Sort results by `usd_volume`, `usd_profit`, `token_volume`, or `token_profit`.
- `startTime` (optional): Filter results starting from a specific time.

**Response**:

```json
{
  "players": [
    {
      "user": "string",
      "creator_fees_usd": "number",
      "usd_profit": "number",
      "usd_volume": "number",
      "token_volume": "number",
      "token_profit": "number"
    }
  ]
}
```

**Example Request**:

```
GET https://api.gamba.so/players?limit=5&sortBy=usd_profit
```
